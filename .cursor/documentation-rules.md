#Documentation rules

##Общие правила

- Для каждого инструмента должно быть короткое и лаконичное описание, показывающая пользователю зачем инструмент нужен и как им воспользоваться.
- Для каждого инструмента должна быть отдельная секция, описывающая, как кастомизировать этот инструмент под проект пользователя.
- В разделе с кратким описанием должна быть ссылка на секцию с описанием кастомизации.

## Правила для readmi

- Краткие описания инструментов должны быть сгруппированы в одну главу
- Описания кастомизации инструментов должны быть сгруппированы в другую главу
